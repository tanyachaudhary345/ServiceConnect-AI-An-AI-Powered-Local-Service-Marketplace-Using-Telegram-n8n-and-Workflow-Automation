# ServiceConnect AI

ServiceConnect AI is a Telegram-based service marketplace automation built using n8n, Google Gemini AI, Google Sheets, and Telegram Bot API.

## Features

* Service provider registration through Telegram
* AI-powered extraction of service details
* Google Sheets database storage
* Customer service search using natural language
* Automatic provider matching
* Instant Telegram responses

## Tech Stack

* n8n
* Google Gemini AI
* Google Sheets
* Telegram Bot API

## Workflow

Provider Registration:

Telegram → AI Agent → JavaScript → Google Sheets

Customer Search:

Telegram → AI Agent → JavaScript → Google Sheets → Match Provider → Telegram Reply

## Example

Provider:

/register Raghu, Carpenter, Patna, 1234567890

Customer:

Need a carpenter in Patna

Response:

Service Provider Found

Name: Raghu
Service: Carpenter
City: Patna
Phone: 1234567890
